GLOBAL ANIMATION RULES (apply everywhere)
=================================================================
1. Install: npm install gsap @gsap/react framer-motion lenis
2. Root layout.tsx: initialize Lenis smooth scroll globally (see code pattern below)
3. Every section entering viewport: GSAP ScrollTrigger fade+slide (opacity 0->1, y:60->0, duration:1, ease:"power3.out")
4. Every page transition: Framer Motion AnimatePresence with mode="wait", 
   0.5s fade+slight vertical shift
5. Every clickable card/button: subtle hover scale (1.03-1.08) + shadow lift
6. Every image: next/image with blur placeholder, lazy loaded except hero images (priority=true)
7. Respect prefers-reduced-motion: wrap all GSAP/Framer animations with a check 
   and disable heavy motion for users who request it
8. Mobile: disable heavy 3D tilt/parallax effects, keep simple fade-ins only 
   (performance + smaller screens don't benefit from tilt effects)

LENIS SETUP CODE (put in app/layout.tsx or a ClientLayout wrapper):
```
'use client'
import { useEffect } from 'react'
import Lenis from 'lenis'

export function SmoothScrollProvider({ children }) {
  useEffect(() => {
    const lenis = new Lenis({ duration: 1.2, smoothWheel: true })
    function raf(time) {
      lenis.raf(time)
      requestAnimationFrame(raf)
    }
    requestAnimationFrame(raf)
    return () => lenis.destroy()
  }, [])
  return <>{children}</>
}
```

NOTE ON 8-SECOND LIMIT: Since Veo generates only 8-10 second clips, always 
add "seamless loop" and "no jump cuts" to every prompt, then in code use:
<video autoPlay muted loop playsInline> — the loop attribute handles 
continuous playback automatically once you have a clean seamless clip. 
For extra safety, use a crossfade technique: duplicate the video element 
and crossfade between two instances at the loop point using Framer Motion 
opacity animation timed to the video duration, eliminating any visible seam.
